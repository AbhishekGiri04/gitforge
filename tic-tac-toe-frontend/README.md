# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- **Interactive Gameplay** - Click on cells to make moves
- **Player Turn Indicator** - Shows whose turn it is (X or O)
- **Win Detection** - Automatically detects wins and draws
- **Reset Functionality** - Reset button to start a new game
- **Responsive Design** - Works on desktop and mobile devices
- **Animated Character** - Cute bouncing cartoon character
- **Floating Clouds** - Beautiful background animations

## How to Play

1. Open `index.html` in your web browser
2. Player X starts first
3. Click on any empty cell to place your mark
4. Players alternate turns (X and O)
5. First player to get 3 in a row wins!
6. Click "Reset Game" to play again

## Winning Conditions

You can win by getting three of your marks in:
- Any horizontal row
- Any vertical column  
- Either diagonal

## Game Rules

- Two players take turns
- Player X always goes first
- Players cannot overwrite existing moves
- Game ends when someone wins or board is full
- Draw occurs when board is full with no winner

## Installation

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start playing!

No additional setup or dependencies required.
